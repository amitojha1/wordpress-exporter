WordPress model exporter for Bustle

== Description ==

WordPress model exporter for Bustle based on the WP JSON API and WP CLI

== Installation ==

1. Activate the plugin
1. Use either the WP JSON API, custom rewrite endpoint, or WP CLI to export your model data

== Frequently Asked Questions ==

= Can I modify the output =

Yes. There are many WordPress filters that can change the models and their values.
